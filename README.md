# CET-4 Archive

英语四级 (CET-4) 备考资料站 — 2024 年至今每场考试的题型分析、考点摘要、自创仿真模拟卷与解题方法论。

**重要声明**:本站所有题目均为站长自创的仿真模拟题,**不复现教育部考试中心的真题原文**。真题请购买官方授权出版物。

## 内容

- 5 场考试页面(2024.6 / 2024.12 / 2025.6 / 2025.12 / 2026.6)
- 每场:题型结构 + 考点话题方向 + 4 部分模拟卷(听力 / 阅读 / 翻译 / 写作)+ 折叠答案
- 共通方法论:写作模板 / 翻译四步法 / 阅读定位 / 听力数字
- 4500 高频词分类索引

## 部署

GitHub Pages / Vercel,纯静态 HTML + CSS + 少量 JS(全部展开按钮)。

样式与折叠机制复用 gaoshu-textbook 工程。
