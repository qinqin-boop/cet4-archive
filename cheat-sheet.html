✍ 写作 (30 min, 120-180 词)

开头In recent years, [topic] has become a widely-discussed issue. From my perspective, [立场].

论据Firstly, ... For example, ... Moreover, ... Statistics show ... In addition, ...

结尾In conclusion, it is essential to ... Therefore, we should ...

替换词very→extremely, important→crucial, many→numerous, help→facilitate, because→due to

衔接举例 For example/递进 Moreover/对比 However/因果 Therefore/总结 In conclusion

→ 完整模板

🔊 听力 (25 min, 25 题)

看选项预判题目朗读前圈关键词 (人名/地点/数字/动词)

抓转折but / however / actually / in fact — 后面是真意

数字13 (thir-TEEN) vs 30 (THIR-ty) 重音区分

同义替换annoyed→frustrated, go to bed→retire for the night

陷阱听到啥选啥 ✗ / 时态主体被换 ✗ / 部分正确 ✗ / quite≠quiet

弃车2-3 秒抓不到关键就猜, 不能卡题

→ 完整技巧

📖 阅读 (40 min, 30 题)

时间分配Section A 7min / B 13min / C 20min

题目先读圈关键词再回原文找

关键词定位数字/专有名词/形容词 → Ctrl+F 思维

排除规律太绝对 (all/never/must) ✗ / 部分对 ✗ / 反向逻辑 ✗ / 无中生有 ✗

词性判断空前 the/a→n., to→v., be→adj./V-ing, 副词→adj.

弃车看不完先放 Section A (单题分值最低)

→ 完整技巧

🌐 翻译 (30 min, 140-160 字)

抓主干找主谓宾, 后半用 with/which/who 接

流水句中文逗号 → 英文 and/but/then 连

补主语中文省略主语, 英文必须补 (常用 it / there be)

时态已经→has +V-ed; 正在→is V-ing; 通常→一般现在

不会就换书法 calligraphy → traditional Chinese writing

高频句式...是...之一; 众所周知; 越来越多的人; 起源于; 在..方面起重要作用

文化词春节/中秋/书法/京剧/四大发明/丝路/一带一路/共同富裕

→ 完整技巧

⚡ 考场最后 10 条

  • 带齐: 身份证 + 准考证 + 2B 铅笔 + 橡皮 + 黑色签字笔 + 手表 (无电子)
  • 提前 30 min 到考场, 听力试音
  • 顺序: 写作 → 听力 → 阅读 → 翻译 (官方顺序)
  • 听力一气呵成, 不能回头改
  • 填涂答案直接到答题卡, 别先记试卷
  • 主观题 (翻译/写作) 字写工整, 别连笔
  • 4 选 1 实在选不出, 排除剩 2 个再蒙
  • 看时间但别看到分神, 戴普通手表
  • 不会的词 / 题目不要空着, 蒙也比 0 分强
  • 425 分及格 = 客观题 60%, 别给自己加戏
