使用流程

  • 1) 进入任一试卷 — 题目按真实考试顺序:听力 → 阅读 → 翻译 → 写作
  • 2) 在线作答 — 客观题 radio 单选,主观题(翻译/写作)textarea 输入
  • 3) 听力 🔊 播放 — 浏览器 Web Speech API 朗读对话(Chrome/Edge/Safari 原生支持)
  • 4) 提交批改 — 底部「✅ 提交批改」按钮,JS 即时判分
  • 5) 查看反馈 — 客观题高亮对错 + 显示得分;主观题展开标准答案对比
  • 6) 重置 — 「🔄 重置」按钮清空作答重新练习
